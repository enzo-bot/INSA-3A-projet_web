# __WORDLE__

## __Installation__

### __Pré-requis__

- _Node.js_
- _Node Package Manager_

Pour installer les dépendances, taper la commande suivante :
```shell
npm install
```

## __Démarrage__

Pour lancer le __serveur HTTP__, taper la commande suivante :
```shell
npm start
```

Rendez-vous ensuiste à l'adresse suivante : http://localhost:3000/
